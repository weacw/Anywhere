API Docs
---
上传信息数据接口（文字数据）
https://weacw.com/uploadinfo.php;
以Get形式上传;
?place="PLACE"&type="TYPE"&descript="DESCRIPT"&version="VERSION"&assetName="ASSETNAME"&thumbnailName="THUMBNAILNAME"

place:地点名称
type:资源类型[360图片、360视频、AB等]
descript:地点介绍
version:资源版本
assetName:资源名称[用于从OSS拉取资源]
thumbnailName:缩略图名称[用于从OSS拉取资源]
---
分页加载信息数据接口（文字数据）
https://weacw.com/getinfo.php;
以Get形式上传;
?page=PAGE
page:查询页数
---
查询信息数据接口（文字数据）
https://weacw.com/searchinfo.php;
以Get形式上传;
?search=PLACE
search:搜索关键词以place为搜索条件
